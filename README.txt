HOW TO RUN TAG

1. Download all the files from the GIT repository
2. Run make program from the terminal in the team401 folder

HOW TO PLAY TAG
1. Hover over player 1 or player two using the navswitch. Player 1 is running away, player 2 is chasing player 1.
2. Push the navswitch to select 1 or 2.
3. Use the navswitch to run away or chase the other player.
4. Player 1 wins if the time runs out (blue LED flashes intensely. Player 2 wins if their LED lands on player 1's LED.
